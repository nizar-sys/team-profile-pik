Microsoft Excel

Microsoft Excel adalah aplikasi spreadsheet yang dikembangkan oleh Microsoft. Aplikasi ini memungkinkan pengguna untuk membuat, mengelola, dan menganalisis data menggunakan tabel, rumus, dan grafik.

  • 1. Aritmatika

    Aritmatika, adalah bagian dari matematika yang mempelajari tentang operasi dasar bilangan (penjumlahan, pengurangan, perkalian, dan pembagian).

    1. Lambang Perhitungan
      Operator Simbol
      Penjumlahan +
      Pengurangan -
      Perkalian *
      Pembagian /
    2. SUM

      Digunakan untuk menjumlahkan nilai pada sel. Menggunakan rumus =SUM(Sel Asal : Sel Tujuan).

    3. Average

      Digunakan untuk mencari rata-rata nilai pada sel. Menggunakan rumus =AVERAGE(Sel Asal : Sel Tujuan).

    4. MIN

      Digunakan untuk mencari nilai terkecil pada sel. Menggunakan rumus =MIN(Sel Asal : Sel Tujuan).

    5. MAX

      Digunakan untuk mencari nilai terbesar pada sel. Menggunakan rumus =MAX(Sel Asal : Sel Tujuan).

    6. Rank

      Digunakan untuk mencari peringkat nilai pada sel. Menggunakan rumus =RANK(Number, Sel Asal : Sel Tujuan, [Order]).

    7. Count

      Digunakan untuk menghitung jumlah sel yang berisi angka. Menggunakan rumus =COUNT(Sel Asal : Sel Tujuan).

    8. CountBlank

      Digunakan untuk menghitung jumlah sel yang kosong. Menggunakan rumus =COUNTBLANK(Sel Asal : Sel Tujuan).

  • 2. IF (Perkondisian)

    Fungsi IF akan menghasilkan sebuah nilai true jika kondisi yang ditentukan terpenuhi dan akan menghasilkan nilai false jika kondisi tidak terpenuhi.

    1. IF

      Digunakan untuk mengevaluasi kondisi dan mengembalikan nilai berbeda berdasarkan kondisi tersebut. Menggunakan rumus =IF(Kondisi, Nilai Jika Benar, Nilai Jika Salah).

    2. Nested IF

      Digunakan untuk mengevaluasi beberapa kondisi dan mengembalikan nilai berbeda berdasarkan kondisi tersebut. Menggunakan rumus =IF(Kondisi1, Nilai Jika Benar, IF(Kondisi2, Nilai Jika Benar, Nilai Jika Salah)).

    3. IF AND

      Digunakan untuk mengevaluasi dua kondisi dan mengembalikan nilai berbeda berdasarkan kondisi tersebut. Menggunakan rumus =IF(AND(Kondisi1, Kondisi2), Nilai Jika Benar, Nilai Jika Salah).

    4. IF OR

      Digunakan untuk mengevaluasi dua kondisi dan mengembalikan nilai berbeda berdasarkan kondisi tersebut. Menggunakan rumus =IF(OR(Kondisi1, Kondisi2), Nilai Jika Benar, Nilai Jika Salah).

  • 3. LEFT, MID, RIGHT

    LEFT, MID, RIGHT adalah fungsi yang digunakan untuk mengambil sebagian dari teks.

    1. LEFT

      Digunakan untuk mengambil sebagian dari teks dari sisi kiri. Menggunakan rumus =LEFT(Teks, Jumlah Karakter).

    2. MID

      Digunakan untuk mengambil sebagian dari teks dari tengah. Menggunakan rumus =MID(Teks, Posisi Mulai, Jumlah Karakter).

    3. RIGHT

      Digunakan untuk mengambil sebagian dari teks dari sisi kanan. Menggunakan rumus =RIGHT(Teks, Jumlah Karakter).

  • 4. VLOOKUP dan HLOOKUP

    VLOOKUP dan HLOOKUP adalah fungsi yang digunakan untuk mencari nilai pada tabel.

    1. VLOOKUP

      Digunakan untuk mencari nilai pada kolom tertentu. Menggunakan rumus =VLOOKUP(Kriteria, Tabel, Kolom, [Benar/Salah]).

    2. HLOOKUP

      Digunakan untuk mencari nilai pada baris tertentu. Menggunakan rumus =HLOOKUP(Kriteria, Tabel, Baris, [Benar/Salah]).

  • 5. Pembuatan Diagram

    Diagram adalah representasi visual dari data. Microsoft Excel menyediakan berbagai jenis diagram yang dapat digunakan untuk memvisualisasikan data.

  • 6. Pivot Table

    Pivot Table adalah tabel dinamis yang digunakan untuk menganalisis data. Pivot Table memungkinkan pengguna untuk mengelompokkan dan merangkum data dengan mudah.
