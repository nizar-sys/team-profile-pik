Microsoft Word

Perangkat lunak untuk membuat, mengedit, dan memformat dokumen teks.

  • 1. Shortcut Pada Microsoft Word

    • Ctrl + A : Memilih semua teks dalam dokumen
    • Ctrl + B : Membuat huruf tebal
    • Ctrl + C : Menyalin teks yang dipilih
    • Ctrl + D : Membuka kotak dialog font
    • Ctrl + E : Membuat teks rata tengah
    • Ctrl + F : Membuka kotak dialog find
    • Ctrl + G : Membuka kotak dialog go to
    • Ctrl + O : Membuka dokumen
    • Ctrl + P : Mencetak dokumen
    • Ctrl + Q : Keluar dari Microsoft Word
    • Ctrl + R : Membuat teks rata kanan
    • Ctrl + S : Menyimpan dokumen
    • Ctrl + T : Membuat paragraf baru
    • Ctrl + U : Membuat garis bawah
    • Ctrl + H : Membuka kotak dialog replace
    • Ctrl + I : Membuat huruf miring
    • Ctrl + J : Membuat teks rata kanan dan kiri
    • Ctrl + K : Membuat hyperlink
    • Ctrl + L : Membuat teks rata kiri
    • Ctrl + M : Membuat paragraf baru
    • Ctrl + N : Membuat dokumen baru
    • Ctrl + V : Menempelkan teks yang disalin
    • Ctrl + W : Menutup dokumen
    • Ctrl + X : Memotong teks yang dipilih
    • Ctrl + Y : Mengulangi perintah terakhir
    • Ctrl + Z : Membatalkan perintah terakhir

  • 2. Drop Cap

    Drop Cap adalah salah satu fitur yang ada di Microsoft Word yang berfungsi untuk memperbesar huruf pertama pada paragraf. Fitur ini biasanya digunakan untuk mempercantik tampilan dokumen.

  • 3. Pembuatan Makalah

    Microsoft Word juga sering digunakan untuk membuat makalah atau laporan. Berikut adalah struktur umum dari sebuah makalah:

    1. Cover
    2. Kata Pengatar
    3. Daftar Isi
    4. Daftar Gambar (Jika Ada)
    5. Daftar Tabel (Jika Ada)
    6. Bab 1 : Pendahuluan
    7. Bab 2 : Pembahasan
    8. Bab 3 : Kesimpulan
    9. Daftar Pustaka

  • 4. Mail Merge

    Mail Merge adalah fitur yang memungkinkan pengguna untuk menggabungkan dokumen utama dengan sumber data lainnya seperti daftar nama, alamat, dan lain-lain. Fitur ini sering digunakan untuk membuat surat undangan, label alamat, dan lain-lain.
